# Bank-Management
The Bank Management System is a command-line application developed in Python with MySQL database connectivity.
This project is designed to simulate the basic operations of a banking system, including account creation, login authentication, balance inquiries, deposits, and withdrawals.

The system allows users to create a new bank account by entering personal details such as name, age, phone number, government ID, and initial deposit amount.
After successful registration, the system automatically generates a unique account number and PIN for secure access.

Existing users can log in using their account number and PIN to perform various banking operations. The application maintains user records in a MySQL database and ensures input validation for secure and error-free transactions.

This project demonstrates the implementation of database management, conditional statements, loops, user authentication, and basic financial transaction logic in Python.
